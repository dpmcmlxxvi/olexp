Simple

This is the simplest example where only an empty map is displayed with the default toolbar and layer manager. Click here for a stand alone example.

Options

This example shows how to pass an optional constructor argument to modify application options. This example moves the location of the navigation and toolbar panels, disables a control, hides the overlays node, changes various settings, and adds an additional context menu to the layer manager items. Click here for a stand alone example.

Layers

This example shows how to add additional map layers and group them using the explorer API. Click here for a stand alone example.

Proj4

Click here for a stand alone example that shows how to use the proj4 library. The example loads a GeoJSON file with a custom coordinate reference system, queries epsg.io for its corresponding definition, adds that definition to proj4, and then loads the GeoJSON layer to the explorer map.
